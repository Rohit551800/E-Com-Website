## ✨ Features

- 🛍️ **Add to Cart** — Add any product to cart with a single click
- 🔢 **Quantity Controls** — Increment or decrement product quantity before adding
- 🗑️ **Remove from Cart** — Remove individual items from the cart
- 💰 **Dynamic Total Calculation** — Real-time Subtotal, GST (Tax), and Final Total
- 💾 **LocalStorage Persistence** — Cart data survives page refresh and navigation
- 🔔 **Toast Notifications** — Instant feedback for add, update, and remove actions
- 📱 **Fully Responsive** — Works seamlessly on mobile, tablet, and desktop
- 🧭 **Multi-Page Navigation** — Home, About, Contact, and Cart pages
- ⚡ **Fast Dev & Build** — Powered by Vite for lightning-fast development

## ⚙️ Installation & Setup

### Prerequisites
- [Node.js](https://nodejs.org/) v16 or higher
- npm (comes with Node.js)

### Steps

**1. Clone the repository**
```bash
git clone https://github.com/Rohit551800/e-com-website.git
cd e-com-website
```

**2. Install dependencies**
```bash
npm install
```

**3. Start the development server**
```bash
npm run dev
```

**4. Open in browser**
```
http://localhost:5173
```

**5. Build for production**
```bash
npm run build
```

> ⚠️ **Note:** Always use a dev server (Vite or Live Server). Opening `index.html` directly via `file://` will cause ES module CORS errors.

## 🧠 How It Works

### 🛒 Cart Flow

```
User clicks "Add to Cart"
         ↓
addToCart.js checks LocalStorage
         ↓
    ┌────┴────┐
  Exists?   New?
    ↓          ↓
Update qty   Push new item
    ↓          ↓
    └────┬────┘
         ↓
Save to LocalStorage ("cartProductsLS")
         ↓
Update cart badge (navbar count)
         ↓
Show toast notification
```

### 💾 LocalStorage Schema

Cart data is stored as a JSON array under the key `"cartProductsLS"`:

```json
[
  {
    "id": 1,
    "quantity": 2,
    "localStoragePrice": 2598
  },
  {
    "id": 3,
    "quantity": 1,
    "localStoragePrice": 1299
  }
]
```

### 💰 Total Calculation

```
Subtotal  =  Sum of (price × quantity) for all cart items
Tax       =  18% (GST)
─────────────────────────────────────────────
Final Total = Subtotal + Tax
```

### 🔔 Toast Notifications

| Trigger | Message | Color |
|---------|---------|-------|
| `showToast("add")` | Product Added To Cart | 🔵 Blue |
| `showToast("alreadyExist")` | Quantity Updated in Cart | 🟠 Orange |
| `showToast("delete")` | Product has been removed | 🔴 Red |
